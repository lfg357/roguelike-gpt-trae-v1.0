## 验收方式

- **验收人**：开发/测试人员

- **验收标准**：设计文档 §22 验收闸口
- **记录方式**：每项勾选通过/不通过，不通过须附截图/描述

## 综合评分（P0闸口）

依据设计文档 §22.2，由验收人打分：

| 维度 | 权重 | 得分（1-5） | 加权 |
|------|------|-------------|------|
| 打击感 | 30% | | |
| 操作手感 | 20% | | |
| 战斗难度 | 20% | | |
| 构筑初体验 | 15% | | |
| 整体意愿 | 15% | | |
| **综合加权** | **100%** | | |

### P0通过标准

- [ ] 综合加权 >= 3.4
- [ ] 无单项 < 2.5
- [ ] 愿意向朋友推荐（Q24）>= 6/10
- [ ] 愿意再玩一次（Q23）>= 60% 打4+
- [ ] 测试者战斗有节奏感
- [ ] 死亡后愿意重试
