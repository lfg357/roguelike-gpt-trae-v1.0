> 覆盖五行系统、层间切换、遗物系统、战斗系统四大模块，共40项用例

---

## 用例编号规则

`TC-{模块}-{序号}`  
模块：FE=五行系统 / FT=层间切换 / RL=遗物系统 / CB=战斗系统
