# Source Art

`combat_vfx_atlas_chroma.png` is the generated flat-key source retained for later
art direction changes. The keyed runtime result is stored in `../atlases/`.

`combat_motion_vfx_chroma.png` is the directional common-action source. Its quick
slash, heavy slash and dash cells are authored facing right; gameplay rotation maps
that baseline directly to the mouse or movement direction.
