```gdscript
{
    id: String          // 唯一标识
    name: String        // 显示名称
```

## 2. 敌人规格

### 2.1 敌人数据结构

```gdscript
{
    name: String       // 显示名称
    hp: float         // 生命值
    damage: float     // 攻击伤害
    speed: float      // 移动速度
    ai: String        // AI类型
    element: String   // 元素属性
    stone: int        // 灵石掉落量
    radius: float     // 碰撞半径
}
```

### 2.2 AI类型（6种）

| AI类型 | 行为模式 | 应对策略 |
|--------|----------|----------|
| idle_wander | 巡逻游荡，受击后转追击 | 主动出击，先手优势 |
| chase | 持续追击近战，接触造成伤害 | 风筝 / 闪避拉开距离 |
| ranged | 保持距离射击，被近身后退 | 突进近身 / 绕掩体 |
| summoner | 定期召唤小怪，本体较弱 | 优先击杀召唤者 |
| charger | 长前摇 + 高伤害 + 短距离突进 | 闪避后反击 |
| shielder | 正面免伤，需绕背或破盾 | 走位绕背 / 破盾技能 |
